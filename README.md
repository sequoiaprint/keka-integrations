#Integration: Keka HR × Machine Planning System
This integration connects Keka HR with Sequoia Print’s machine planning and daily operations workflow.

#Objective
To sync employee attendance, shifts, and leave data from Keka directly into the machine utilization and production planning modules.

#Key Features
1. Real-time Data Sync: Automatically fetch operator attendance and shift details from Keka.
2. Smart Scheduling: Reflect operator availability in daily machine plans and job allocations.
3. Performance Mapping: Link attendance and work hours to actual machine output for efficiency tracking.
4. Automated Reports: Generate daily summaries combining HR data with production KPIs.
5. Seamless Workflow: Unified dashboard for both manpower and machine utilization.

#Outcome
This integration enables data-driven scheduling, minimizes idle time, and improves transparency between HR and production operations.
